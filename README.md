# ZacSkillHub

A personal collection of reusable Codex skills.

## Structure

Each skill lives in its own top-level folder and includes a required `SKILL.md` file. Skills may also include supporting `scripts/`, `references/`, `assets/`, or `agents/` folders when needed.

## Current Skills

- `jira-csd`: Create, read, comment on, attach files to, and resolve Jira CSD issues through Agora Lab APIs.
- `ai2earn`: Reusable skill content for the AI2Earn workflow.

## Maintenance

Keep skills focused, concise, and self-contained. Prefer reusable scripts for deterministic workflows, and validate skill metadata before merging changes.
